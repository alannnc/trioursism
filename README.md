# How to use

In a terminal, run the following commands:

```
codesign -s - .bitcoin/bin/bitcoind
```

```
npm install
npm run btc:start
```

In another terminal run:

```
npm run icp:start
```

Open another terminal and run:

```
npm run icp:deploy
```

or

```
npm run icp:deploy:local
```
